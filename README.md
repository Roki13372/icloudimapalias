# iCloud Hide My Email Manager

**Удобный инструмент для генерации iCloud алиасов, мониторинга кодов и проверки почты.**

Автоматически создаёт Hide My Email алиасы, ловит верификационные коды и отправляет их в Telegram, а также позволяет быстро искать письма за любой период.

---

## ✨ Основные возможности

- **Генерация алиасов** — реальная генерация и бронирование через Apple Hide My Email API
- **Watcher кодов** — отслеживает новые письма с кодами подтверждения и мгновенно отправляет их в Telegram
- **Checker писем** — поиск писем за выбранный период (2, 4, 24, 48 часов) по ключевому слову
- **Экспорт профилей** — в TXT файл или сразу в Telegram
- **Удобный веб-интерфейс** — всё в одном месте: таблица профилей, генерация, watcher, checker и живой лог
